| 函数            | 精度        | 说明                                          |
| --------------- | ----------- | --------------------------------------------- |
| time()          | 秒          | 获取系统时间                                  |
| ftime()         | 秒 + 毫秒你 | 获取当前的时间和日期(deprecated)              |
| gettimeofday()  | 秒 + 微秒   | 获取当前时间, 以微秒精度存储在timeval结构体中 |
| clock_gettime() | 秒 + 纳秒   | 获取高精度的时间, 存储在timespec结构体中      |
